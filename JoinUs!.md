# Join SREA
## Security Research & Engineering Architecture

SREA is a modular security-focused framework designed for research, analysis, and engineering across multiple programming languages.

We welcome contributors interested in:
- Security research
- Reverse engineering
- Malware analysis
- Systems programming
- Tooling development (Python, Java, C, C++, C#, Go, Rust, Haskell, Shell, Assembly)
